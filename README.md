## Setup

Clone the repository

```bash
  git clone https://github.com/Tofazzal-2443/Quiz-App.git
```
Move to the desired folder

```bash
  cd \Quiz-App
```
To run the app, simply write

```bash
  flutter pub get
```
```bash
  flutter run
```
